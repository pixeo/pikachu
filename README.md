# Pikachu (Peek-at-you)

A simple site launch checklist checker.

![](https://media.giphy.com/media/xuXzcHMkuwvf2/giphy.gif)

## Installation

Download or clone the repository and run

```bash
composer install
```

## Usage

``` php
php pikachu check http://somesite.tld/sitemap.xml
```
